PERIMETRE DE L'AUDIT
================================================================================

Surface exposee :
  - Endpoint POST /api/llm/generate-quiz/ (contenu utilisateur -> LLM)
  - Champ texte /upload (saisie libre)
  - Upload PDF /upload (contenu extrait par pypdf puis envoye au LLM)

Vecteurs de risque identifies :
  1. Prompt injection via champ texte
  2. Prompt injection cachee dans un PDF
  3. Exfiltration du prompt systeme
  4. Manipulation des reponses generees
