2. RESULTATS DE VALIDATION - MUST HAVE
================================================================================

US-01 - Inscription email
  Statut    : VALIDE
  Test      : Creation d'un compte avec email + mot de passe
  Resultat  : Auto-login apres inscription, email de validation dans les logs
  Date      : 29/06/2026

US-02 - Connexion email / mot de passe
  Statut    : VALIDE
  Test      : Login avec email + mdp corrects, deconnexion
  Resultat  : Token JWT stocke, pages protegees inaccessibles sans token
  Date      : 29/06/2026

US-06 - Saisie cours en texte (>= 200 car.)
  Statut    : VALIDE
  Test      : Saisie de 576 car. (Biologie - Cycle de Krebs), soumission
  Resultat  : Validation 200 car. fonctionnelle, quiz genere avec succes
  Date      : 29/06/2026

US-07 - Upload PDF (<= 5 Mo)
  Statut    : NON TESTE - Report Sprint 2
  Motif     : Capacite Sprint 1 a 70% (post-perturbation P1)
              Test PDF non prioritaire vs. validation F1 + saisie texte
  Date      : -

US-08 - Generation 10 QCM automatique
  Statut    : VALIDE (avec reserve)
  Test      : Generation via llama3.1:8b (LLM local)
  Resultat  : 10 QCM generes avec succes - MAIS latence mesuree : ~3 min
              Le critere d'acceptation "< 60 s" n'est pas respecte en local CPU
              Decision prise : LLM_BACKEND=groq (2-5 s) - cf. ADR-0001
  Date      : 29/06/2026

US-09 - Repondre aux 10 QCM
  Statut    : VALIDE
  Test      : Navigation avant/arriere, soumission finale du quiz
  Resultat  : Parcours complet fonctionnel
  Date      : 29/06/2026

US-10 - Voir son score /10
  Statut    : VALIDE
  Test      : Affichage score apres soumission
  Resultat  : Score /10 affiche correctement
  Date      : 29/06/2026

US-11 - Detail des corrections
  Statut    : VALIDE
  Test      : Affichage bonnes/mauvaises reponses apres quiz
  Resultat  : Corrections presentes sur la page de resultats
  Date      : 29/06/2026

US-12 - Historique des quiz
  Statut    : VALIDE
  Test      : Navigation vers /history apres quiz
  Resultat  : Liste des quiz avec date et cours visible
  Date      : 29/06/2026
