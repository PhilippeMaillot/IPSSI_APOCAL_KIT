1. CONTEXTE DE LA PERTURBATION
================================================================================

Le PO signale en debut de Sprint 5 que la Release 1 (17h45 mercredi) doit etre
conforme RGPD avant demonstration. Deux droits utilisateur sont obligatoires
(reglement UE 2016/679) :

  - Art. 17 (droit a l'effacement) : l'utilisateur doit pouvoir supprimer
    son compte et toutes ses donnees associees.
  - Art. 20 (droit a la portabilite) : l'utilisateur doit pouvoir telecharger
    ses donnees personnelles dans un format lisible.
  - Art. 13 (obligation d'information) : des pages legales doivent etre
    accessibles depuis le site (mentions legales, politique de confidentialite,
    CGU, cookies).

Ces obligations existaient deja dans le cadrage initial mais leur statut
MoSCoW ne les rendait pas prioritaires. La perturbation P4 les rend MUST HAVE.

3. VERDICT POUR RELEASE 1 (mercredi 17h45 STRICT)
================================================================================

La Release 1 est invalide sans ces trois elements :
  1. US-05 fonctionnel (suppression compte + donnees)
  2. US-18 fonctionnel (bouton export JSON)
  3. 4 pages legales completes et accessibles

Note : US-05 est deja code dans le kit EduTutor IA (backend + frontend).
Note : US-18 necessite un endpoint backend a implementer (TASK-27).
Note : Les pages legales sont des squelettes vides dans le frontend (TASK-26).
